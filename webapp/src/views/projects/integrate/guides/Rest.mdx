## Exporting data

```
{{{apiUrl}}}/api/project/export/jsonZip?ak={{{apiKey}}}
```

or with curl

```shell-session
curl "{{{apiUrl}}}/api/project/export/jsonZip?ak={{{apiKey}}}" \
--output data.zip
unzip data.zip
rm data.zip
```
