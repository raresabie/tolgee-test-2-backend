## Install the packages

You will need `@tolgee/vue` to use `<T>` component or `TolgeeMixin` with `$t` function.

```
npm install @tolgee/vue
```

## Setup your environment

Add this to your `.env.development.local`.

```dotenv
VUE_APP_TOLGEE_API_URL={{{apiUrl}}}
VUE_APP_TOLGEE_API_KEY={{{apiKey}}}
```

## Use TolgeeProvider

Wrap your whole app with Tolgee provider.

```html
<template>
  <TolgeeProvider :config="config"> ... </TolgeeProvider>
</template>

<script>
  import { TolgeeProvider } from '@tolgee/vue';

  export default {
    components: { TolgeeProvider },
    data() {
      return {
        config: {
          apiUrl: process.env.VUE_APP_TOLGEE_API_URL,
          apiKey: process.env.VUE_APP_TOLGEE_API_KEY,
        },
      };
    },
  };
</script>
```

## Use Tolgee!

```html
<template>
  <T keyName="translation_key" />
</template>

<script>
  import {T} from "@tolgee/vue";

  ...
</script>
```

or

```html
<template>
  <div :title="$t('translation_key')">Hello</div>
</template>

<script>
  import { TolgeeMixin } from '@tolgee/vue';

  export default {
    mixins: [TolgeeMixin],
  };
</script>
```
